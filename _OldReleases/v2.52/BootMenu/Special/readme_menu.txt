The Boot Menu code and the custom font were created by RBSC.

There are 2 main flavors of the boot menu:

\BOOTCMFC.BIN		- standard Boot Menu, suitable for most of MSX computers
\BOOTCMFC.ALT		- standard Boot Menu, suitable for most of MSX computers with different music
\SPECIAL\BOOTCMFC.BIN	- custom Boot Menu for Korean and Arabic MSX computers; it can be also used with any other MSX
\SPECIAL\BOOTCMFC.ALT	- custom Boot Menu for Korean and Arabic MSX computers with different music; it can be also used with any other MSX

See the readme.txt file for more info.
